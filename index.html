What is the Ionic Framework?

  • Frontend framework for developing hybrid mobile apps with HTML5 & AngularJS for both iOS & Android platforms.
  • Requires Node.js (hopefully pre-installed on all machines)
  • For iOS apps, requires OS X, Xcode & command line tools
  • For Android requires the Android SDK
  • Without Xcode or Android SDK you can still create view & debug many apps in a web browser.
